# Chris's Discount Project Maker — v0.30.0

Static Microsoft Project-inspired planner that runs on GitHub Pages.

## v0.30.0 changes

- Added a fuller Microsoft Project-style Gantt Chart Format ribbon.
- Adds Format, Columns, Bar Styles, Gantt Chart Style, Show/Hide, and Drawings groups.
- Adds Text Styles, Gridlines, Layout, Insert Column, Column Settings, Custom Fields, Critical Tasks, Slack, Late Tasks, Task Path, Baseline, Slippage, a 16-style Gantt gallery, Outline Number, Project Summary Task, Summary Tasks, and Drawing commands.
- Wires available commands to existing app behavior: gridline visibility, text-size toggle, column width presets, baseline command, summary task show/hide, critical/late highlighting, and Gantt style color changes.
- Placeholder toasts remain for later modules such as custom fields, slack, task path highlighting, slippage, project summary task rendering, and drawing objects.

## Use

Open `index.html` locally or host the folder on GitHub Pages. Use the restored MS Project-style schedule grid for real planning, dependency links, predecessor/successor columns, baselines, progress, actuals, resources, XML import/export, and CSV export. Use Task, Project, View, and Gantt Chart Format tabs for the fuller MS Project-style command surface.
